# Create: Nuclearics Mod
This is the GitHub repository of the Create: Nuclearics mod, A Create Addon Mod that adds Nuclear
functionality such as Nuclear Reactors and Radiation.

*Although Mod is still WIP, and most of the features isn't implemented yet.
But the Mod Source code from this repository should be buildable into a jar file.

This mod is a Forge 1.18.2 Mod. (Because 1.18.2 is the current modding version in 23/3/2023)
